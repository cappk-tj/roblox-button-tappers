# Button Tappers

An Incremental / Clicker game where players gain cash, buy upgrades, rebirth and progress.

### Tech
- Roblox Studio
- Luau

### Features
- Cash System — Players gain cash by clicking buttons with varying multipliers
- Upgrades — Players can buy upgrades to increase their base clicking cash amount
- Multipliers — Players can increase their cash multipliers using premium currency (Diamonds) or through rebirthing
- Rebirths — Players gain multiple stat multipliers in exchange for restarting.
- Data Stores — Player data saves across different sessions
- UI Feedback — UI feedback whenever players interact with certain features
- Stats — Players can keep track of different stats such as lifetime cash earnt, join date, playtime, etc.

### The Process
I used to play cookie clicker a lot a fair bit ago and tried to recreate it within Roblox! Cookie clicker but 3D and with a ton of random / cool features I added along the way. This project took me 6 months to complete and is definitely my most polished & biggest project I have made as of 18th April 2026. Overall, pretty happy with how the game turned out!

### What I could have improved
- Clicking mechanic — Whenever a player clicked a button to gain cash the client would send a remote to the server and do a bunch of validation checks, anti cheat, etc. Which caused a lot of server lag when a server was full of players and everyone was clicking at the same time. Players on lower-end devices suffered the most from this. I could have instead ??? Idk tbh
- UI Design — Some UI elements could have been made better by using a consistent stylized design across all UI.
